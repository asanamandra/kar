My Projects i've built from my own learning

Node.js Shopping Cart Repo
Web App Hosted on Heroku

The shopping cart is a simple check out web application built with Node.js. Node.js utilizes JavaScript for server side scripting using npm tools like Express, Passport and Request packages. The shopping cart app is built with MongoDB database and utilizes mongoose schema structure. For the front end, the web app uses Bootstrap elements and ejs (https://ejs.co/) for DOM manipulation. For internal APIs within the app for data flow, JSON is being used and can be tested with Postman (https://www.getpostman.com/). The methods associated with API requests are GET, POST, DELETE and UPDATE. /users handles all users. /users/specific-username handles specific users.

Daily Cook Mobile App Repo (Private)

Daily Cook is a Android based mobile application built with Ionic Framework. The app aims to give a platform to home cooks who wish to sell their homemade food online. The app uses google location to track food available within a certain radius. Users can either buy or sell food through the app.


Oculus Go Headset App Repo
Oculus Go App Website

Oculus Go Headset App - University of South Carolina Capstone Project. Client wanted an Oculus Go VR App for the Oculus Go headset which would give a tour of the USC Campus in VR. All of the controller mappings and C# scripting were done using Unity. 360 Pictures of the campus were taken using the Garmin Virb 360 camera. App is then loaded onto the headset along with Audio Descriptions.
